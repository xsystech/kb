# Системные требования

Стандартный виртуальный сервер архитектуры: **x86/x86-64.**

Количество ядер vCPU: **2**

Количество оперативной памяти: **2 ГБ**

Свободное место на жестком диске: **10 ГБ**

Поддерживаемые операционные системы для установки:

| Ubuntu 18, 20, 22 / Debian 10, 11, 12                     |
| --------------------------------------------------------- |
| RHEL 8, 9 / CentOS 8, 9/ RockyLinux 8 / Oracle Linux 8, 9 |
| Astra Linux (CE, SE) 1.6, 1.7                             |
| РЕД ОС 7.2, 7.3 / AlterOS 8 / Rosa Enterprise Linux       |
| ОС АЛЬТ 9, 10                                             |

В операционной системе должен быть установлен Docker и Docker Compose.&#x20;

Рекомендуемые версии:

* Docker Compose: версия 2.17 и выше
* Docker: версия 23.0 и выше

Поддерживаемые среды серверной виртуализации для автоматизации аварийного восстановления с помощью XRM:

| [oVirt 4](https://ovirt.org/) / [RHEV 4](https://www.redhat.com/en/technologies/virtualization/enterprise-virtualization) / [OLVM 4](https://www.oracle.com/cis/virtualization/)                                             |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [HOSTVM 4.\*](https://pvhostvm.ru/) / [zVirt 3.\*](https://orionsoft.ru/zvirt) /  [РЕД Виртуализация 7.\*](https://www.red-soft.ru/ru/content/rv) / [ROSA Virtualization 2.х](https://www.rosalinux.ru/rosa-virtualization/) |
