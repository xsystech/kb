# История версий

**2026-01-29: Версия 1.2.1 Update 3 (**[**Документация**](https://kb.x-rm.ru/dokumentaciya/dokumentaciya/xrm-versiya-1.2.1)**)**

* Исправлена ошибка отображения списка модулей.
* Обновлены версии внутренних компонентов: Python версии 3.10, Docker Compose с поддержкой Docker API 1.44
 
**2024-07-25: Версия 1.2.1 Update 2 (**[**Документация**](https://kb.x-rm.ru/dokumentaciya/dokumentaciya/xrm-versiya-1.2.1)**)**

* Добавлен плагин для автоматизации аварийного восстановления VDI инфраструктуры
* Доработаны сценарии установки, исправлены ошибки

**2023-11-20: Версия 1.2.1 Update 1 (**[**Документация**](https://kb.x-rm.ru/dokumentaciya/dokumentaciya/xrm-versiya-1.2.1)**)**

* В установочный сценарий добавлена возможность изменения учетной записи и пароля администратора.
* Доработана локализация элементов WEB-интерфейса.
* Улучшен алгоритм генерации сценариев восстановления, содержащих разные типы хранилищ.

**2023-09-13: Версия 1.2**

* Добавлена поддержка генерации сценариев восстановления для хранилищ iSCSI и FC.
* Добавлен интерактивный мастер установки и управления сервисами XRM.
* Исправление ряда визуальных ошибок в WEB-интерфейсе.

**2023-07-14: Версия 1.1**

* Добавлен WEB-интерфейс управления.
* Добавлена поддержка генерации сценариев восстановления для хранилищ NFS.
* Исправлена логика генерации пути хранилищ для предотвращения некорректного добавления при восстановлении.

**2023-02-10: Версия 1.0**

* Добавлена поддержка запуска автоматизированных сценариев восстановления.
* Запущена база знаний.
