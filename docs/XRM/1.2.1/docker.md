⚠️ Установка Docker и Docker Compose может потребовать прав администратора на вашей ОС. Если вы не обладаете необходимыми правами, обратитесь к администратору системы для выполнения данных действий.


**Рекомендуемые версии Docker и Docker Compose:**

Для оптимальной и стабильной работы с **XRM** рекомендуем использовать следующие версии **Docker** и **Docker Compose**:

1. Docker Compose: версия 2.17 и выше
2. Docker: версия 23.0 и выше
